Features

  • Identify the object or scene name
    • Supports the identification of 100,000 common objects and scenes such as animals, plants, commodities, buildings, landscapes, animations, foodstuffs, and public figures. The interface returns the names of major and sub-categories.
  • Get encyclopedia information
    • Supports obtaining the encyclopedia information corresponding to the picture recognition result, and the interface returns the encyclopedia entry URL, picture and summary description, and can choose whether to return the encyclopedic information.

Application scenario

  • Image content analysis and recommendation
  • Photographing
  • Photographing fun marketing
