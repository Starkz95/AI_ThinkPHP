Features

  • Face similarity comparison
    • Compare the similarity of two faces in the picture and return the similarity score
  • Support 4 picture types
    • Support for life photos, ID photos, ID card photos, and face patterns with 4 picture types
  • Picture quality control
    • Analyze the occlusion, ambiguity, illumination intensity, attitude angle, integrity, and size of the face in the picture, and return an accurate similarity score based on the output of the image that meets the quality standard.

Application scenario

  • Financial remote account opening
  • Service personnel supervision
  • Wisdom brush face attendance
  • Hotel self check-in
